# 项目简介

## 核心功能
- 使用 Django 框架进行后端开发
- 集成 Pyecharts 进行数据可视化
- 应用随机森林和线性回归两种机器学习算法
- 数据采集与预处理模块
- 模型训练与预测模块
- 结果展示与交互模块

## 适用场景
- 能源管理
- 能源消耗预测
- 电力系统优化

## 技术特点
- 基于 Django 框架，提供稳定的后端支持
- 结合 Pyecharts，实现数据的高效可视化
- 随机森林和线性回归算法的应用，提高预测准确性
